# Building and Installation

## Installation Procedure

### 1. Configuration

ขั้นตอนแรกของการติดตั้ง คือการกำหนดค่าโครงสร้างซอร์สโค้ด สำหรับระบบ และเลือกตัวเลือกที่ต้องการ ทำได้โดยการเรียกใช้สคริปต์ `configure` สำหรับการติดตั้งแบบค่าเริ่มต้น ให้ป้อนคำสั่งต่อไปนี้:

```
ubuntu:~/postgresql-18.3$ ./configure --with-systemd
```

สคริปต์นี้จะทำการทดสอบหลายอย่าง เพื่อกำหนดค่าสำหรับตัวแปรต่างๆ ที่ขึ้นอยู่กับระบบ และตรวจจับความผิดปกติของ OS และสุดท้ายจะสร้างไฟล์ เพื่อบันทึกสิ่งที่ค้นพบ

### 2. Build

```
# To start the build, type either of
ubuntu:~/postgresql-18.3$ make
```

### 3. Regression Tests

ทดสอบเซิร์ฟเวอร์ที่สร้างใหม่ก่อนติดตั้ง `Regression Tests` เป็นชุดทดสอบเพื่อตรวจสอบว่า PostgreSQL ทำงานบนเครื่องได้ตามที่ต้องการ:

```
ubuntu:~/postgresql-18.3$ make check
```

### 4. Installing the Files

ในการติดตั้ง PostgreSQL ให้ป้อนคำสั่งต่อไปนี้:

```
ubuntu:~/postgresql-18.3$ sudo make install
```

ขั้นตอนนี้จะติดตั้งไฟล์ลงในไดเร็กทอรีที่ระบุไว้ในขั้นตอนที่ 1 โดยปกติแล้ว ต้องทำขั้นตอนนี้ในฐานะผู้ใช้ root

### 5. Cleaning

หลังจากการติดตั้ง สามารถเพิ่มพื้นที่ว่างในดิสก์ได้ โดยการลบไฟล์ที่สร้างขึ้น ด้วยคำสั่ง `make clean` แต่จะเก็บรักษาไฟล์ที่สร้างโดย `./configure` ไว้ เพื่อให้สร้างทุกอย่างขึ้นมาใหม่ได้ด้วยคำสั่ง `make` ในภายหลัง

## Uninstallation

หากต้องการยกเลิกการติดตั้ง ให้ใช้คำสั่ง `make uninstall` อย่างไรก็ตาม คำสั่งนี้จะไม่ลบ เช่น `data directory`
