# Linux on Windows with WSL

## Installing Linux Distribution

```
# list of available Linux distributions
wsl --list --online

# By default, the installed Linux distribution will be Ubuntu.
wsl --install [Distro]

# To set the default Linux distribution
wsl --set--default [Distro]
```

## Export & Import a distribution

```
# Terminates the specified distribution.
wsl --terminate ubuntu-24.04

# Exports the distribution to a tar file.
wsl --export ubuntu-24.04 $env:userprofile/Downloads/ubuntu.tar

# Imports the specified tar file as a new distribution.
wsl --import ubuntu-24.04 $env:userprofile/AppData/Local/Packages/Ubuntu $env:userprofile/Downloads/ubuntu.tar
```

## Installing the software packages

```
ubuntu:~$ sudo apt update && \
sudo apt install -y bzip2 build-essential \
pkg-config libicu-dev bison flex libreadline-dev \
zlib1g-dev libsystemd-dev liblz4-dev vim-gtk3 \
liburing-dev

# เพิ่มเติมสำหรับ man
sudo apt install -y libxml2-utils docbook-xml docbook-xsl xsltproc

# เพิ่มเติมสำหรับ pgbouncer
sudo apt install -y libevent-dev libssl-dev pandoc
```
