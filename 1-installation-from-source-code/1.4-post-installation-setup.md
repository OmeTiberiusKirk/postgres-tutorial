# Post-Installation Setup

## Environment Variables

หากติดตั้งลงใน /usr/local/pgsql-17.9 ควรเพิ่ม /usr/local/pgsql-17.9/bin ลงใน PATH

ให้เพิ่มข้อความต่อไปนี้ ลงในไฟล์ start-up ของ shell เช่น `~/.bash_profile`, `~/.profile` (หรือ `/etc/profile` หากต้องการให้มีผลกับผู้ใช้ทั้งหมด):

```
# /etc/profile
export PATH=/usr/local/pgsql-17.9/bin:$PATH
ubuntu:~$ source /etc/profile
```

เพื่อให้ระบบสามารถค้นหาเอกสารคู่มือการใช้งาน (man documentation) ได้ ค้องเพิ่มบรรทัดต่อไปนี้ลงในไฟล์เริ่มต้นของเชลล์

```
# /etc/profile
export MANPATH=/usr/local/pgsql-17.9/share/man:$MANPATH
ubuntu:~$ source /etc/profile
```
