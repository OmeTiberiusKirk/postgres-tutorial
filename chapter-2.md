# Chapter 2. The SQL Language

## Concepts

- PostgreSQL เป็นระบบจัดการฐานข้อมูลเชิงสัมพันธ์ (RDBMS) ซึ่งหมายความว่า เป็นระบบที่ จัดการข้อมูลที่จัดเก็บในรูปแบบความสัมพันธ์
- Relation คือคำศัพท์ทางคณิตศาสตร์ ที่โดยพื้นฐานแล้วหมายถึงตาราง (table)

| เชิงคณิตศาสตร์ | เชิงฐานข้อมูล (SQL) |
| -------------- | ------------------- |
| Relation       | Table               |
| Tuple          | Row / Record        |
| Attribute      | Column              |
| Domain         | Data Type           |

- **table** คือชุดของ **rows** ที่ถูกตั้งชื่อ
- **row** ใน **table** จะมีชุด **columns** ที่ถูกตั้งชื่อเช่นกัน
